Application de gestion de tâches

Une application mobile pour la gestion collaborative des tâches conçue avec Flutter avec une architecture moderne.... - Backend : Node.js avec Express, MongoDB pour la base de données. - Frontend : Flutter pour une expérience utilisateur fluide et réactive.
Fonctionnalités clés : - Authentification sécurisée (JWT)
- Création, modification et suppression de tâches
- Partage de tâches entre utilisateurs
- Notifications en temps réel


Ce projet m'a permis de renforcer mes compétences en développement mobile, en gestion de bases de données et en intégration d'API RESTful.
