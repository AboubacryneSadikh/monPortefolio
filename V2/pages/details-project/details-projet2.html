Projet #02

Application de gestion
de présence

Description

Application desktop de gestion de présence développée avec JavaFX pour le client lourd et Spring Boot pour l'API REST.

Stack technique

Backend

Spring Boot (API REST) + MySQL

Frontend

Angular + Angular Material

Fonctionnalités

  • Authentification sécurisée (JWT)
  • Système de pointage employés
  • Historique des présences et absences
  • Notifications automatiques
  • Dashboard administrateur

Ce projet m'a permis d'améliorer mes compétences en développement desktop, architecture REST et gestion des états avec Angular.
