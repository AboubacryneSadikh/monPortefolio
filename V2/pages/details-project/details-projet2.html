Application de gestion de présence

Description :

Application desktop de gestion de présence développée avec JavaFX et Spring Boot.

  • Backend : Spring Boot (API REST) + MySQL
  • Frontend : Angular + Angular Material

Fonctionnalités :

  • Authentification sécurisée (JWT)
  • Système de pointage
  • Historique des présences
  • Notifications
  • Dashboard admin

Ce projet m’a permis d’améliorer mes compétences en développement desktop et web.
