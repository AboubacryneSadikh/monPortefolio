Application de gestion
de projet

Description

Une application mobile de gestion de projet collaboratif développée avec Flutter pour le frontend et Laravel pour le backend.

Stack technique

Backend

Laravel (API REST) + MySQL

Frontend

Flutter — moderne et responsive

Fonctionnalités

  • Authentification sécurisée (JWT)
  • Gestion des projets et tâches
  • Attribution des tâches aux membres
  • Suivi des progrès en temps réel

Ce projet m'a permis de renforcer mes compétences en API REST, architecture MVC et bases de données relationnelles.
