Application de gestion de projet

Description :

Une application mobile de gestion de projet collaboratif développée avec Flutter pour le frontend et Laravel pour le backend.

  • Backend : Laravel (API REST) + MySQL
  • Frontend : Flutter moderne et responsive

Fonctionnalités :

  • Authentification sécurisée (JWT)
  • Gestion des projets et tâches
  • Attribution des tâches
  • Suivi des progrès

Ce projet m'a permis de renforcer mes compétences en API REST et bases de données.
